# Lets learn kotlin
## Compiling kotlin from CMD line
```
	kotlinc file.kt -include-runtime -d file.jar
```

+ Run with
```
	java -jar file.jar
```
+ Magic it in the makefile

```kotlin
	var num_1 = 10
```
